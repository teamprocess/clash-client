# Clash Monorepo

## Workspace

- apps/mac: Electron macOS client
- apps/web: Web frontend
- packages/tsconfig: Shared TypeScript config presets
- packages/eslint-config: Shared ESLint flat config presets

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
